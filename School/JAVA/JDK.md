# JDK
### JDK
* 기본 도구
> * java: 실제 자바를 실행시키는 도구
> * javac: 소스코드 컴파일
> * javadoc: API 문서 생성
> * jdb: 자바 소스 디버깅
> * jar: Java Archaive(JAR) 생성 도구
> * appletviewer: 애플릿 간단하게 실행하는 도구
* 보안 도구
> * jarsigner: jar 파일 서명 생성 도구
> * policytool: 보안 정책 관리
* 국제화 도구
> * native2ascii: 텍스트 유니코드 변경 도구
* 모니터링 도구
> * VisualVM: JVM을 모니터링하기 위한 JMX 지원 그래픽 도구
