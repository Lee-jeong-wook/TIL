# BFS(너비 우선 탐색)
### What?
- 루트노드(혹은 임의의 노드) 에서 인접한 노드를 먼저 탐색하는 것
### For?
최단 경로나 임의의 경로를 찾아야 할 때
### Characteristic
- 재귀적 X
- Dijkstra와 비슷
- Queue 사용
### Process
- 깊이가 1인 노드 방문 후 깊이 2인 노드 방문
- 더이상 방문할 곳이 없다면 종료
##### Detail
- 시작 노드 방문 후 체크, 큐에 삽입
- 큐에서 꺼낸 노드와 인접한 노드들 방문, 인접한 노드가 없다면 노드를 꺼냄(dequeue)
- Queue에 방문한 노드 삽입
- Queue가 소진되면 종료
